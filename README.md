Health Assist is a web-based healthcare application designed to **digitalize patient data management** and provide healthcare professionals with a **centralized, easy-to-use dashboard** for accessing medical information.

The application helps doctors and medical staff efficiently view patient details, monitor vital signs, review diagnosis history, and analyze lab results, reducing dependency on manual records and improving overall workflow efficiency.


## Project Overview
In many healthcare environments, patient data is scattered across paper records or multiple systems.  
**Health Assist** solves this problem by bringing essential patient information into **one interactive web interface**, enabling faster access and better clinical decisions.


## Key Features
**Patient Data Management**
View patient personal and medical details in one place


**Vital Signs Tracking**
Monitor key health indicators such as temperature, heart rate, and blood pressure


**Diagnosis History**
Access previous diagnoses and treatment records


**Lab Results Review**
View laboratory test results through a clean interface


**Interactive Dashboard**
User-friendly dashboard for quick navigation and insights


**API Integration**
Fetch and display real-time or mock healthcare data using APIs


**Basic UI & UX Design**
Clean layout focused on usability and clarity

 ## Purpose & Impact
 The purpose of this project is to:
